Quick Reference

  • Edit your config directly in the browser — supports both settings.json (JSON) and run.py (Python) formats
  • Validate checks syntax (JSON or Python) before saving
  • Save creates a .bak backup, validates, then writes
  • Revert reloads the file from disk, discarding unsaved changes
  • Export run.py generates an upstream-compatible run.py from settings.json
  • Available strategies: SMART, MOST_VOTED, HIGH_ODDS, PERCENTAGE, SMART_MONEY, HISTORICAL, KELLY_CRITERION, CONTRARIAN, MOMENTUM, VALUE_BET, WEIGHTED_AVERAGE, UNDERDOG, NUMBER_1–NUMBER_8
  • Discord muting: muted_channels=["streamer"], muted_events_per_channel={"streamer": [Events.BET_LOSE]}, global_muted_events=[Events.GAIN_FOR_WATCH]
  • Secrets: In JSON mode, prefix values with $ to read from environment variables (e.g. "$TWITCH_PASSWORD")
